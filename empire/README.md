# EMPIRE CORE (Citadel System)

Monorepo workspace hosting:

- `apps/citadel-core` (central control hub)
- `apps/referrals-live` (referral marketplace)
- `apps/theunitedstates-site` (news + SEO engine)
- `apps/tmack48-media` (media/video/music hub)
- `packages/shared` (cross-site automation + monetization primitives)

## Quick start

```bash
cd empire
npm install
npm run dev:all
```

## Production build

```bash
npm run build:apps
```
